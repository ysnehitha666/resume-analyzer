# 📄 Resume Analyzer

An AI-powered web app that analyzes resumes and provides instant feedback using Google Gemini AI.

## Features
- 🤖 AI-powered resume analysis
- 🎯 ATS Score Check
- 📌 Missing Skills Detection
- 💡 Suggestions for improvement
- 📜 History of past analyses (SQLite database)
- ⚖️ AI-powered resume comparison
- 🗑️ Delete history entries (full CRUD)

## Tech Stack
- Frontend: HTML, CSS, JavaScript
- Backend: Python, Flask
- AI: Google Gemini API
- PDF Parsing: PyMuPDF
- Database: SQLite
